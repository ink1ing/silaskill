## Notes

- `apple-home-without-gateway` is a reference pack for controlling Apple Home devices from a local Mac without a gateway.
- The pack is split into two layouts:
  - direct toggle
  - automation stack
- It includes core dependency notes and operational guidance for reproducible local testing.
